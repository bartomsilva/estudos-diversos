Web Design Responsivo

Fonte: Wikipédia, a enciclopédia livre.

Web Design Responsivo (responsive web design) é uma abordagem para criar sites onde a experiência do usuário é otimizada independente da forma a qual ele acessa o website, facilitando a navegação e scrolling em qualquer caso.

Para fins de demonstração a fonte deste parágrafo é fixa e igual a 16px

Para fins de demonstração a fonte deste parágrafo é fixa e igual a 24px

Em outras palavras, independente de usar um desktop ou um dispositivo móvel, independente da resolução ser 1024x780 ou 320x480, o conteúdo de um web site responsivo vai se adequar a este espaço, sem perder informações e nem a capacidade de transmitir o necessário ao usuário.

Há diversas técnicas que podem ser aplicadas para atingir um design responsivo com mais facilidade. Os três principais são o uso de media queries, o layout fluído e as imagens flexíveis.
